# CSS DOOM Static Lab

A static browser-based DOOM/CSS experiment deployed from GitHub using static hosting.

## Overview

CSS DOOM Static Lab is a lightweight frontend hosting project built to practice static site deployment, GitHub version control, and clean technical documentation.

The goal is to keep the project simple:

HTML + CSS + JavaScript  
GitHub Repository  
GitHub Pages or Cloudflare Pages  
Public Website  

No backend, VM, Docker container, or database is required for the current version.

## Project Goals

- Build a static browser-based web project
- Host it through GitHub Pages or Cloudflare Pages
- Document the deployment process
- Capture screenshots for portfolio proof
- Prepare the project for future custom domain routing

## Deployment Plan

### Phase 1: Local Project Setup

Create the static site structure and test the files locally.

### Phase 2: GitHub Workflow

Use Git branches, commits, pushes, and pull requests to manage changes.

### Phase 3: Static Hosting

Deploy the site using GitHub Pages first.

### Phase 4: Custom Domain

Move or mirror the deployment to Cloudflare Pages and connect a custom subdomain later.
