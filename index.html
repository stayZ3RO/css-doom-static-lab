About This Project

CSS DOOM Static Lab is a portfolio project focused on static web hosting, GitHub-based deployment, and clean documentation. The goal is to keep the architecture simple while still showing practical web deployment skills.

Architecture

Local Files → GitHub Repo → Static Hosting → Public Website

Documentation Goals

  • Explain why this project does not need a VM or backend.
  • Document the GitHub workflow from local changes to public deployment.
  • Capture screenshots for portfolio proof.
  • Prepare future Cloudflare Pages and custom domain notes.

Built as a static web hosting lab for GitHub portfolio documentation.
